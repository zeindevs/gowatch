# gowatch

<p align="center">
    File whatcher with custom triggered command when file has been changed.
</p>

## Features

- Custom triggered command
- Watching folder and subfolder
- Filter by file extension

## Usage

```sh
gowatch -cmd 'echo "file changed"' -dir views -ext .html
```
